# Plano Estratégico: Site da Agência Web (Marseille/Aubagne)

## Visão Geral
Landing page moderna para agência web na região de Marseille/Aubagne, França.
Foco em captar clientes locais (construção, saúde, profissionais liberais, comércio).

**Objetivo principal:** Gerar leads qualificados (sem mostrar preços no site)

**Público-alvo:**
- Empresas de construção (400+ na região)
- Profissionais de saúde (médicos, dentistas, fisioterapeutas)
- Profissionais liberais (advogados, contadores)
- Comércios locais (restaurantes, salões, oficinas)
- Serviços à pessoa (limpeza, jardinagem, babás)

**Idiomas:** FR (principal), EN, PT, ES

# ESTRATÉGIA DE CAPTAÇÃO DE CLIENTES

## Fluxo de Conversão

```
Visitante → Lê Hero → Identifica-se com Problema →
Vê Diferencial Tech → Explora Serviços →
Entende Processo → Confia (Social Proof) →
Preenche Formulário ou WhatsApp → LEAD
```

1. **Hero** - CTA principal (formulário)

3. **Após Processo** - "Réserver ma consultation gratuite"
4. **Final** - Formulário completo
5. **Flutuante** - Botão WhatsApp sempre visível
